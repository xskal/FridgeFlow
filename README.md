# 🥘 FridgeFlow — Умный планировщик питания

**FridgeFlow** — это веб-приложение для планирования меню, учёта продуктов в холодильнике и автоматической генерации списков покупок. Помогает экономить время на готовке, избегать просрочки продуктов и оптимизировать расходы на питание.

> 🎯 **Для кого?** Для тех, кто хочет питаться разнообразно, не выбрасывать еду и тратить на готовку меньше времени и денег.

## ✨ Возможности

- **Регистрация и авторизация** — личный кабинет для каждого пользователя
- **Каталог рецептов** — удобные карточки с описанием, ингредиентами и временем готовки
- **Виртуальный холодильник** — учёт продуктов со сроками годности и предупреждениями о просрочке
- **Планирование меню на неделю** — календарь с завтраками, обедами и ужинами
- **Умный список покупок** — автоматическая генерация на основе плана питания с учётом остатков из холодильника
- **Аналитика с графиками** — визуализация расходов, КБЖУ и топ-продуктов (Plotly)

## 🚀 Как запустить проект локально

### 1. Клонируйте репозиторий:
```bash
git clone https://github.com/xskal/FridgeFlow.git
cd FridgeFlow
```

### 2. Создайте и активируйте виртуальное окружение:
# Windows
```bash
python -m venv venv
venv\Scripts\activate
```

# Linux/Mac
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установите зависимости:
``` bash
pip install -r requirements.txt
```

### 4. Создайте файл .env в корне проекта:
```bash
SECRET_KEY=ваш-секретный-ключ
DEBUG=True
ALLOWED_HOSTS=localhost,127.0.0.1
DATABASE_URL=sqlite:///db.sqlite3
```

### 5. Выполните миграции:
```bash
python manage.py migrate
```

### 6. Создайте суперпользователя (для доступа в админку)
```bash
python manage.py createsuperuser
```

### 7. Запустите сервер разработки:
```bash
python manage.py runserver
```

### 8. Откройте в браузере:
Главная страница: http://127.0.0.1:8000/
Админ-панель: http://127.0.0.1:8000/admin/
